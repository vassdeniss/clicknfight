Click 'n Fight: The Fantasy Clicker RPG



Album with Screenshots!

Description

Have you ever wanted to play an RPG but not one of those big open world time consuming games? Ever wanted to click away and enjoy something small? You like clicker games but cookie clicker got boring? Try Click 'n Fight!

I started this game as my first C# project ever and originally I wanted to make a clicker game like cookie clicker but after some days I changed it to an RPG game. The game features fighting, shops, magic spells, weapons, crafting and many more!

Engage in the Endless Mode after you finish the final boss or give a hand at the New Game + Mode!

I really hope everyone enjoys this little game and I am open to critisism as I would like to improve my programming.

Well what are you waiting for? Give it a try if you can!


Install Instructions

- Download setup.exe
- Run setup.exe
- Give it admin rights (for installing and saving under C:/ drive)
- Select install directory
- Select if you want desktop shortcut
- Run from desktop or install directory
